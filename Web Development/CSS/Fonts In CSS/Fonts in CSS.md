###### Font Family အကျဥ်းချုပ်

font တစ်ခုဆိုတာ <mark style="background: #FFB86CA6;">သီးခြားသတ်မှတ်ထား</mark>တဲ့ <mark style="background: #FFF3A3A6;">typeface ကိုယူသုံးထား</mark>တဲ့ <mark style="background: #BBFABBA6;">Fonts Style Group </mark>တစ်ခု။
font တစ်ခုမှာတင် မတူညီတဲ့ font style တွေ ပါတယ်။
- regular font, bold font, light font, ..etc
- font တစ်ခုမှာ ပါဝင်တဲ့ fontတွေ အားလုံးက <mark style="background: #BBFABBA6;">တူညီတဲ့ design concept typeface</mark> တစ်ခုအောက်မှာပဲ ဖန်တီးထားတာ

font family ဆိုတာက fontတွေကို အုပ်စုဖွဲ့၊ groupပုံစံနဲ့ အသုံးပြုတာကိုပြောတာ
font family ကို အဓိကအားဖြင့်
1. Generic Font Family
2. Specific Font Family ဆိုပြီး နှစ်မျိုးခွဲလို့ရတယ်။

Specific Font Family - ဆိုတာက ဒီ Design အတွက် ဒီ font တစ်မျိုးဲကိုဘဲ သုံးမယ်ဆိုပြီး **တိတိကျကျ**တစ်မျိုးထဲဘဲ သုံးတာကိုပြောတာ။

Generic Font Family - ဆိုတာက <mark style="background: #BBFABBA6;">ပုံစံဆင်တူ (similar) ဖြစ်တဲ့ <span style="color:rgb(192, 0, 0)">typeface‌</span></mark> <span style="color:rgb(192, 0, 0)"></span>တွေသုံးထားတဲ့ fontတွေကို အုပ်စုဖွဲ့သုံးတာ။ Web Dev လောကမှာတော့ Generic font family ကသာ <span style="color:rgb(255, 255, 0)">အသံုးများ</span>
- Common Generic Fonts - serif, sans-serif, monospace...

ဘာလို့ Generic Font family ကို အသုံးများလဲဆိုရင် userရဲ့ စက်ထဲမှာ web dev ဘက်က သတ်မှတ်ပေးလိုက်တဲ့ fontတွေရှိမနေတာ၊ အကြောင်းကြောင်းကြောင့် မပြနိုင်တာမျိုးဆိုရင် fallbackအနေနဲ့ userရဲ့စက်ထဲက ပုံစံဆင်တူတဲ့ font ကိုယူပြ။
ဒီတော့ ဒီဇိုင်းအရ သိသိသာသာ <span style="color:rgb(255, 255, 0)">ပြောင်းလဲမှုမရှိ</span>
