
html မျာ " < " နဲ့ " > " က HTML elements တွေ‌ရေးဖို့ သုံးတယ်။ ဒီတော့ browser က " < " နဲ့ ">" ကိုမြင်ရင် element ရေးဖို့ဘဲလို့ထင်ပြီး ဘာသာပြန်တယ်။ ဒီတော့ greater than & less than signs တွေကို browser မှာ ပြချင်ရင် ဘယ်လို‌ရေးရမလဲဆိုတော့ <mark style="background: #D2B3FFA6;">html symbols တွေသုံး</mark>ပြီး ရေးရမယ်။

Greater than - &gt;
Less than - &lt;

```
<p> HTML says &lt;h1&gt; Hello World &lt;/h1&gt; </p>

```

output in browser = HTML says < h1 > Hello World < /h1 >
